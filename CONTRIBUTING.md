# Contributing to BIOQUA

Thank you for your interest in improving BIOQUA!

## How to Contribute

1. **Fork** this repository
2. **Create a branch** for your feature: `git checkout -b my-feature`
3. **Make your changes** — keep the credit header in all source files
4. **Test** your changes
5. **Submit a Pull Request** with a clear description

## Rules

- **Keep the credit header** — every `.py` and `.ino` file must include the
  BIOQUA co-author credit header at the top. Run `python python/verify_credits.py`
  to check.
- **One feature per PR** — keep pull requests focused
- **Test before submitting** — make sure the existing system still works
- **Document your changes** — update the manual if you add new features

## Ideas for Contributions

- New sensor plugins (dissolved oxygen, nitrate, fluoride)
- Improved AI models (higher accuracy, more bacteria classes)
- Mobile app improvements (iOS support, better UI)
- Additional learning modules
- Translation to other languages
- 3D printable enclosure designs
- Improved data visualization in reports

## Code Style

- Python: follow PEP 8
- Arduino: use camelCase for functions, UPPER_CASE for constants
- Comments: explain *why*, not *what*
- Keep it simple — this is a student project

## Questions?

Open an issue on GitHub and we'll help you get started.
